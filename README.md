# web-security-labs

> Hands-on web security labs, CTF writeups, and vulnerability research notes.
> Built as part of an active journey into penetration testing and red teaming.

---

## About

This repository documents practical web security work covering real-world attack techniques,
OWASP Top 10 vulnerabilities, and structured lab walkthroughs. Every entry is based on
legal, ethical environments (TryHackMe, HackTheBox, PortSwigger Web Academy).

---

## Topics Covered

| Category                  | Examples                                      |
|---------------------------|-----------------------------------------------|
| Injection                 | SQL Injection, Command Injection              |
| Broken Access Control     | IDOR, Privilege Escalation                    |
| XSS                       | Reflected, Stored, DOM-based                  |
| Authentication Flaws      | Brute Force, Session Hijacking                |
| Security Misconfiguration | Directory Listing, Default Credentials        |
| SSRF / XXE                | Server-Side Request Forgery, XML Injection    |
| Recon & Enumeration       | Subdomain discovery, Directory fuzzing        |
